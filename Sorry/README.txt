=============== HOW TO USE ===============

1. CREATE A NEW FOLDER on your Desktop called "apology-website"

2. SAVE 4 FILES inside that folder:
   • index.html    (First page - copy the first code)
   • sorry.html    (Second page - copy the second code)
   • README.txt    (This file)

3. HOW TO RUN:
   • Open the "apology-website" folder
   • DOUBLE-CLICK "index.html"
   • It will open in your web browser
   • Click "Read My Message →" to go to second page
   • Click "← Back to First Page" to return

4. TROUBLESHOOTING:
   • If pages don't open: Right-click → "Open With" → Chrome/Firefox
   • If button doesn't work: Check both files are in SAME folder
   • To edit: Right-click file → "Open With" → Notepad/TextEdit

5. CUSTOMIZE:
   • Change text: Edit between <p> and </p> tags
   • Change colors: Edit "background: #667eea" (hex color codes)
   • Add your name: Replace "@anujbuilds" with your name

6. SHARE ONLINE:
   • Upload all files to: https://app.netlify.com/drop
   • Drag and drop the folder
   • Get a free website URL!
